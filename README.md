# freelance-web
This is Ashish Darji's software development consulting website.

## Tools Being Used
* Gulp
* Bower
* NPM
* BrowserSync
* Bourbon
* FontAwesome
* BootStrap
* jQuery
* AnimateCSS
* VelocityJS
* Etc...


## Launch Website

cd into the directory.


```
gulp updateTools 
```

```
gulp updatePackages
```

```
gulp build
```

```
gulp watch
```
