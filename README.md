# freelance-web
This is Ashish Darji's software development consulting website.

## Tools Being Used
* Gulp
* Bower
* NPM
* BrowserSync
* Bourbon
* FontAwesome
* BootStrap
* jQuery
* AnimateCSS
* VelocityJS
* Etc...


## Launch Website

cd into the directory.

1. 
```
gulp updateTools 
```
2.
```
gulp updatePackages
```
3.
```
gulp build
```
4.
```
gulp watch
```
