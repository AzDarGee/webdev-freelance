# Lucid Labs
Lucid Labs Web App

## Tools Being Used
* Gulp
* Bower
* NPM
* BrowserSync
* Bourbon
* FontAwesome
* BootStrap
* jQuery
* AnimateCSS
* VelocityJS
* Etc...


## Launch Website

cd into the directory.

1. ```gulp updateTools```
2. ```gulp updatePackages```
3. ```gulp build```
4. ```gulp watch```
